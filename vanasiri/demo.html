ðŸ§ª How to Test the Website

1. Shopping Experience

  1. Go to the Shop page to browse products
  2. Use filters to narrow down products by category, price, or size
  3. Search for specific plants using the search bar
  4. Click "Add to Cart" on any product
  5. View your cart by clicking the cart icon in the header
  6. Modify quantities or remove items from cart
  7. Proceed to checkout to see the complete flow

2. Contact Form

  1. Navigate to the Contact page
  2. Try submitting the form with invalid data to see validation
  3. Fill in valid information and submit to see success message
  4. Test phone number formatting and character counter

3. Responsive Design

  1. Resize your browser window to see responsive behavior
  2. Test on mobile devices or use browser dev tools
  3. Check that all elements adapt properly to different screen sizes

4. Cart Persistence

  1. Add items to cart
  2. Refresh the page or navigate to different pages
  3. Verify that cart items are still there
  4. Close and reopen the browser to test localStorage
